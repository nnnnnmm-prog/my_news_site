# Новостной сайт на Django

## Описание
Это новостной сайт, где пользователи могут:
- Регистрироваться и входить
- Добавлять новости с картинками
- Редактировать и удалять новости

## Технологии
- Python 3.10
- Django 5.2
- Bootstrap 5
- SQLite

## Как запустить
1. Скачать проект
2. Установить зависимости: `pip install -r requirements.txt`
3. Применить миграции: `python manage.py migrate`
4. Запустить сервер: `python manage.py runserver`
